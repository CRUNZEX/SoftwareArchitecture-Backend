# Software Architecture: Queue Management System

### Description
- Framework: NestJS
- Database: PostgreSQL

### Requirement
- .env

```
NODE_ENV = production

POSTGRES_HOST = <IP>
POSTGRES_PORT = <PORT>
POSTGRES_USER = <USERNAME_DATABASE>
POSTGRES_PASSWORD = <PASSWORD_DATABASE>
POSTGRES_DATABASE = softarch

JWT_SECERT = <STRING_RANDOM>

```

- config-firebase.json

### Initial

- Install Package
```
npm i
```

- Run Project
```
npm start
```
